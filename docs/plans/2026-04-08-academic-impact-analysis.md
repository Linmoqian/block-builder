# 学术影响分析

## 竞品差异化

现有深度学习可视化工具分为四类，没有工具同时覆盖所有维度：

| 工具类型 | 代表 | 构建 | 训练 | 3D | 代码生成 |
|----------|------|------|------|-----|---------|
| 架构可视化 | Net2Vis, NN-SVG | 否 | 否 | 否 | 否 |
| 训练监控 | TensorBoard, W&B | 否 | 仅监控 | 否 | 否 |
| 教学工具 | TF Playground, GANLab | 部分 | 有限 | 否 | 否 |
| 可视化编程 | DeepVisual, VELCRO, DeepBlocks | 是 | 部分 | 否 | 部分 |
| **神经网络工坊** | **本项目** | **是** | **是** | **是** | **是** |

三个独特创新点：
1. **3D 空间表达** — 跳跃连接、多分支拓扑用 Z 轴层叠展示
2. **类型约束端口** — tensor/scalar/optimizer/data_source 类型匹配
3. **完整闭环** — 构建 → 生成代码 → 训练 → 推理

## 教育层面

深度学习教育痛点及对应解决方案：

| 痛点 | 对应解决 |
|------|----------|
| 抽象概念难理解 | 积木 = 概念，理论符号映射为可操作实体 |
| 代码门槛高 | 拖拽替代编码，自动生成 PyTorch 代码 |
| 缺乏即时反馈 | 端口类型约束 + 实时训练曲线 |
| 架构设计缺乏直觉 | 3D 可视化拓扑 + 预置模板对比 |

Georgia Tech GANLab 研究已证明可视化交互显著缩短理解复杂模型的时间。

## 研究可及性

- 快速原型验证：拖拽组装 → 秒级生成可运行代码
- 架构探索：直观对比不同网络拓扑
- 教学辅助：研究者课堂演示工具

## 学术产出路径

### 首选目标

- **SIGCSE TS**（ACM CS 教育研讨会）— 录取率约 30%
- **IEEE VIS EduVis Workshop** — GANLab 曾在此发表

### 高匹配期刊

- **IEEE TVCG** — Net2Vis、GANLab 发表于此
- **ACM TOCE** — CS 教育顶级期刊
- **Computers & Education**（Elsevier）— 教育技术综合期刊

### 论文核心论点

1. 3D 可视化对理解复杂网络拓扑的效果（对照实验：2D vs 3D）
2. 类型约束端口系统对减少架构错误的帮助（用户研究）
3. 可视化构建 + 代码生成闭环的学习效果评估

### 评估建议

学术界评审重视用户评估。建议工具完成后设计 20-30 人对照实验（工具 vs 传统编码），测量学习速度、错误率、理解深度。
